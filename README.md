# FocusQuest - Habit Tracker

A web app to track daily habits, visualize progress, and build consistency.

## Live Demo
https://focus-quest-wwen.onrender.com/

## Features
- User authentication (register/login)
- Add and track daily habits
- Check off habits each day
- View stats and completion rates
- 30 day progress chart
- Current streak tracking

## Built With
- Python / Flask
- SQLite
- HTML, CSS, JavaScript
- Chart.js
